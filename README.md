Smart Build Hub
A web-based platform to simplify construction project management by connecting Admins, Contractors, and Homeowners in one system.

📌 Overview
Smart Build Hub is designed to make construction work more organized and transparent.
It allows users to manage projects, assign contractors, and track progress easily.

🎯 Key Features
🔐 Secure Login & Authentication
👨‍💼 Admin Dashboard to manage users & projects
👷 Contractor panel to apply & update work
🏠 Homeowner panel to create & track projects
📊 Real-time project updates
💻 User-friendly interface

🧩 Modules

👨‍💼 Admin (adminapp)
Manage users (Contractor & Homeowner)
Approve and monitor projects
Full system control
👷 Contractor (contractorapp)
View available projects
Apply for projects
Update work progress
🏠 Homeowner (homeownerapp)
Create new projects
Assign contractors
Track project status
⚙️ Main App (mainapp)
Handles authentication and common functionalities

🛠️ Tech Stack
Backend: Python (Django)
Frontend: HTML, CSS, Bootstrap
Database: SQLite

Objective
1.The main objective of Smart Build Hub is to solve common problems in the construction process such as fraud, 
lack of transparency, and poor communication.
2.To reduce the risk of fraud by middlemen or brokers by directly connecting homeowners and contractors
3.To provide a trusted platform where users can find and hire verified contractors easily
4.To improve transparency by allowing real-time project tracking
5.To help people living in other cities or countries monitor their home construction remotely
6. To make the construction process more efficient, secure, and reliable
